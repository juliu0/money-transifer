# Money Transfer App

A Flutter application for sending, receiving, and investing money.

## Project Structure

```
lib/
├── main.dart                 # Application entry point
├── app.dart                  # MyApp widget configuration
├── core/
│   └── theme.dart           # Application theme
└── screens/
    └── home/
        └── home_screen.dart # Home screen widget
```

## Features

- Material Design 3 UI
- Modular architecture
- Clean separation of concerns

## Getting Started

1. Ensure you have Flutter installed
2. Run `flutter pub get` to install dependencies
3. Run `flutter run` to start the application

## Development

This project follows Flutter best practices with a well-organized directory structure:
- **core/**: Contains shared resources like themes, constants, and utilities
- **screens/**: Contains UI screens organized by feature
